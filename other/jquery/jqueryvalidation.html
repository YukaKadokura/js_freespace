JQueryValidation

JQueryValidationを使用すると、クライアント向けのバリデーションを簡単に実装できます。

バリデーションしたいフォームに対して、下記のように指定します。

                    
                            $('フォームのIDやクラスを指定').validate({
                                rules : {
                                    フォーム部品の名前:{
                                        required:true
                                    }
                                }
                            })

rulesオプション

# オプション オプションに設定する値 オプションの説明
1 required true / false 必須項目
2 email true / false Emailのフォーマットになっているかチェック
3 equalTo ”要素名” 要素の値と一致しているか
