JQueryValidation

JQueryValidationを使用すると、クライアント向けのバリデーションを簡単に実装できます。

バリデーションしたいフォームに対して、下記のように指定します。

                
                        $('フォームのIDやクラスを指定').validate({
                            rules : {
                                フォーム部品の名前:{
                                    required:true
                                }
                            }
                        })
